#Overview
This project analyzes an online retail transaction dataset to uncover sales patterns, customer behavior, and key business insights.
The goal is to demonstrate data cleaning, exploratory data analysis (EDA), and customer segmentation techniques using Python.
#Dataset
The dataset contains transactional data from an online retail store, including information about products, customers, quantities, prices, and countries.

Dataset source:
https://archive.ics.uci.edu/ml/datasets/online+retail
